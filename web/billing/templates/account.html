Get Started

Run Gamwich on your home server or Raspberry Pi with a single command.

1. Pull the Docker image

docker pull dukerupert/gamwich:latest

2. Run it

docker run -d -p 8080:8080 -v gamwich-data:/data dukerupert/gamwich:latest

3. Open in your browser

Visit http://your-server:8080 to set up your family.
