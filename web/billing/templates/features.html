Features

Everything your household needs, designed for touchscreens and phones.

Glanceable Dashboard

See everything at a glance — the time, today's weather, upcoming events, pending chores, and grocery counts. Designed to be readable from 6 feet away.

  • Live weather from Open-Meteo
  • Today's events and chore summary
  • Rewards leaderboard
📈

Family Calendar

A shared calendar built for families. Day and week views with color-coded events make it easy to see who has what and when.

  • Recurring events (daily, weekly, monthly, yearly)
  • Edit single or all occurrences
  • All-day and timed events
📅

Chore Lists & Rewards

Assign chores to family members, track completions, and motivate everyone with a built-in points and rewards system.

  • Organize by area (kitchen, yard, etc.)
  • Recurring schedules (daily, weekly, monthly)
  • Points per chore, redeemable for custom rewards
✅

Grocery Lists

Add items and they're automatically sorted by store aisle. Check off as you shop, and the list updates everywhere in real time.

  • Auto-categorization by item name
  • Grouped by category for efficient shopping
  • Clear checked items with one tap
🛒

Notes & Message Board

Leave messages for the family, pin important notes, and let old ones auto-expire. Priority colors make urgent messages stand out.

  • Pin important notes to the top
  • Color-coded priority (urgent, normal, fun)
  • Auto-expiring notes with optional dates
📝

Real-Time Sync

Every change is pushed instantly to all connected screens via WebSocket. No polling, no delays.

  • Instant updates across all devices
  • Scoped refreshes — only affected sections update
  • Auto-reconnect with exponential backoff
⚡

Touchscreen Kiosk Mode

Purpose-built for always-on wall displays. Automatic idle screensaver, night-time dimming, and burn-in prevention keep your screen looking great.

  • Idle screensaver with clock and next event
  • Night mode with scheduled dimming
  • Pixel-shift burn-in prevention
💻

Mobile Companion

Install Gamwich as a Progressive Web App on any phone or tablet. Responsive design works from 375px phones to 1920px kiosk displays.

  • Install as app on iOS and Android
  • Responsive from phone to wall display
  • Network-first with offline resilience
📱
